2. TYPOGRAPHY — Trendy Modern Fonts
Replace all current fonts with this pairing:
Headings and display text: "Clash Display" (import from Fontshare — https://api.fontshare.com/v2/css?f[]=clash-display@700,800&display=swap). This is the font used by the most premium SaaS products right now — sharp, confident, slightly editorial. Weight 700-800, letter spacing -0.03em, giving headlines a tight luxury feel.
Body and UI text: "Satoshi" (import from Fontshare — https://api.fontshare.com/v2/css?f[]=satoshi@400,500,600&display=swap). Clean, geometric, modern. Weight 400 for paragraphs, 500 for feature descriptions, 600 for labels and captions.
Apply Clash Display to: main hero headline, section headings, pricing plan names, pricing amounts, and the final CTA heading. Apply Satoshi everywhere else. This pairing is what separates a generic SaaS page from something that feels like it was designed by a top agency.
